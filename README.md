## 1. Abstract
  This project started as a basic project for segmentation of Putamen tissue in MSA patients. The image segmentation algorithm is based on autoencoder. Autoencoder is an unsupervised learning ai algorithm, and takes the form of transforming the dimension of input data and then restoring it back to its original state. By applying this, segnet allows the user to obtain the desired data by restoring only a specific part of the image, not the original image.
